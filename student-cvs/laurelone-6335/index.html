Comandos y Alias en Terminal

Comandos

A continuación se muestran algunos comandos útiles a emplear en la Terminal.

Comando Descripción
mkdir <dir> Crea una nueva carpeta
rm -R <dir> Elimina una carpeta con su contenido
cd [folder] Cambia de directorio
pwd Muestra el directorio del trabajo
ls Lista el nombre de los archivos y subcarpetas del directorio actual

Alias

De los alias mostrados en el video son practicos los siguientes dos:

  • Para evitar eliminar permanente los archivos alias rm="trash"
  • El comando de clear se suele usar mucho alias cl="clear"
